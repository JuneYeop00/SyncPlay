# Extension

OTT 플랫폼에서 현재 시청 중인 콘텐츠 정보를 추출하기 위한 크롬 확장프로그램입니다.

## 소개
이 확장프로그램은 사용자가 OTT 사이트에서 보고 있는 콘텐츠의 제목, 회차 정보, 진행률 등의 데이터를 추출해 팝업 화면에서 확인할 수 있도록 구성되어 있습니다.

플랫폼별로 추출 로직을 분리하여 관리하고 있으며, 공통 제어는 `popup.js`에서 담당합니다.  
현재는 여러 OTT 플랫폼의 시청 정보 추출 기능을 중심으로 구현되어 있고, 추후 대시보드 및 서버 연동 기능까지 확장할 예정입니다.

## 주요 기능
- 현재 접속 중인 OTT 플랫폼 감지
- 콘텐츠 제목 및 회차 정보 추출
- 시청 시간 및 진행률 확인
- 팝업 창에서 추출 결과 출력
- 플랫폼별 추출 로직 분리 관리

## 파일 설명
- `manifest.json` : 크롬 확장프로그램의 권한, 실행 파일, 기본 설정을 정의하는 파일
- `popup.html` : 확장프로그램 팝업 화면 UI 파일
- `popup.js` : 팝업 동작을 제어하고 현재 페이지에서 콘텐츠 정보 추출을 실행하는 메인 스크립트
- `disney-bridge.js` : 디즈니플러스 페이지 전용 정보 추출 스크립트
- `netflix-bridge.js` : 넷플릭스 페이지 전용 정보 추출 스크립트
- `tving-bridge.js` : 티빙 페이지 전용 정보 추출 스크립트
- `watcha-bridge.js` : 왓챠 페이지 전용 정보 추출 스크립트
- `wave-bridge.js` : 웨이브 페이지 전용 정보 추출 스크립트
- `coupang-bridge.js` : 쿠팡플레이 페이지 전용 정보 추출 스크립트
- `README.md` : 확장프로그램 설명 문서

## 구조 설명
이 확장프로그램은 플랫폼별로 정보 추출 방식을 분리한 구조입니다.

- `popup.js`는 메인 제어 파일로서 현재 페이지를 확인하고 알맞은 추출 로직을 실행합니다.
- 각 `*-bridge.js` 파일은 해당 OTT 플랫폼의 DOM 구조나 재생 정보에 맞춰 제목, 회차, 진행률 등을 추출합니다.
- 이렇게 분리함으로써 플랫폼별 수정이 필요할 때 전체 코드를 크게 건드리지 않고 개별 파일만 수정할 수 있습니다.

## 실행 방법
1. 크롬 브라우저에서 `chrome://extensions` 에 접속합니다.
2. 우측 상단에서 개발자 모드를 활성화합니다.
3. `압축해제된 확장 프로그램 로드`를 클릭합니다.
4. 프로젝트의 `extension` 폴더를 선택합니다.
5. OTT 페이지에 접속한 뒤 확장프로그램을 실행하여 정보를 확인합니다.

## 주의사항
- OTT 사이트 구조가 변경되면 일부 추출 기능이 정상적으로 동작하지 않을 수 있습니다.
- 플랫폼마다 페이지 구조가 달라 추출 정확도에 차이가 있을 수 있습니다.
- 현재는 개발 및 테스트 단계이며, 지원 플랫폼은 계속 확장될 예정입니다.
